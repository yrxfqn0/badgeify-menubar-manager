## Badgeify 是什么？一句话说清楚

**Badgeify 是一款 macOS 菜单栏图标管理工具**，核心功能是：把任意 Mac 应用的图标添加到菜单栏，同时实时显示该应用的未读消息数量（角标）。

它解决的是一个很具体的痛点：macOS 默认只在 Dock 图标上显示角标。如果你把 Dock 隐藏掉，或者正在全屏模式下工作，根本看不到有没有新消息进来。Badgeify 把这些通知"搬"到了菜单栏，让你始终都能瞄到。

支持 Mail、信息、Slack、Teams、WhatsApp、Discord……只要是用了 macOS 通知系统的应用，基本都能接入。系统要求是 macOS 11 Big Sur 及以上版本。

## 核心功能拆解：它能帮你做什么

### 实时通知角标

在菜单栏显示应用的未读消息数。有三种角标样式可选：文字数字、圆点、空心圆圈。每个应用还可以单独设定强调色，比如把 Slack 的角标设成绿色，把邮件设成红色，扫一眼就能分清楚谁有消息。

这个功能对"隐藏 Dock 党"来说尤其好用。根据 Reddit 社区 r/macapps 的讨论，不少用户反映把 Dock 彻底隐藏后，靠 Badgeify 菜单栏图标来监控通知，工作专注度反而提升了——需要看消息才去看，不需要的时候干扰更少。

### 分组管理

把多个应用合并成一个菜单栏图标，显示合计的未读数。比如把 Slack、Teams、邮件归进一个"工作消息"组，菜单栏只占一个位置，角标数是三者的总和。点开才会看到分组内各应用的独立数据。

这对菜单栏空间寸土寸金的 MacBook 用户来说很有价值——屏幕越小越需要这个。

### 智能隐藏与透明化

无未读消息时，图标自动变透明或完全隐藏。没有通知的应用不占视线，有消息才冒出来提醒你。菜单栏整洁，信息优先级也更清晰。

### 快速启动

点击菜单栏图标可以直接打开对应应用，不用回 Dock 或用 Spotlight。对于隐藏了 Dock 的用户，这相当于把 Dock 的启动功能也搬进了菜单栏。

### 自定义图标

专业版和终身版支持上传自定义图标（支持 SVG 格式），把原来那个不好看的应用图标换掉，菜单栏整体风格也能统一。

## 如何使用 Badgeify：从安装到设置的完整步骤

### 第一步：下载安装

前往数码荔枝下载页，或直接下载限制功能的免费版先体验。安装完成后打开应用，会要求授权 macOS 辅助功能权限——这是 Badgeify 读取其他应用 Dock 图标角标数据所必须的，不会收集或外传你的个人信息。

### 第二步：添加应用到菜单栏

打开 Badgeify 侧边栏，点击「+」按钮，从你已安装的应用列表中选择想要监控的应用。选完后，该应用的图标会出现在菜单栏右侧。

### 第三步：配置通知显示

点击菜单栏上的图标，进入该应用的设置。打开「显示通知」开关，选择角标样式（文字/圆点/圆圈），按需设置强调色。设置后，有未读消息时角标会实时更新。

### 第四步：设置分组（专业版/终身版）

在侧边栏点击「新建分组」，把相关应用拖入同一组。分组图标会合并显示所有成员的未读数之和，整洁又高效。
